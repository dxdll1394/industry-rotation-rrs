七、数据局限性

  1. RS（相对强度）非绝对价格。RS下降不=价格下跌，只是跑输大盘
  2. 行业等权聚合，个股可能不同于行业整体
  3. 147个采样点，间隔5个交易日，日内波动未被捕捉
  4. 45日+样本量偏少，统计意义有限
  5. 过去3年规律不代表未来，市场结构可能变化

八、采样间隔影响（补充说明）

2026-05-17 进行了全梯度对比（间隔3~20，步长2），关键发现：

  • ↓4+占比不受采样间隔影响，稳定在10-15%，是固有统计特征
  • ↓3逆转率呈双峰模式——间隔7（21日）峰值20%，间隔11~13（33~39日）低谷5~6%
  • 存在两种自然逆转周期：1.5周的短周期和3.5周的中周期调整
  • 5日间隔是统计可靠性和实用性的最佳平衡
