## 一、算法说明

### 趋势分段
从RS轨迹（5日采样）识别连续同向运动段，每段至少2期（10个交易日）。

### 层级合并
小幅逆势波（振幅 < 主波振幅×33%）被合并到周围的大趋势中，消除噪声。

### 主浪评分
上升浪按 `振幅 × √交易天数` 评分，最高分即为该行业的历史最强主升浪。

### 位置分类

| 阶段 | 含义 | 判断条件 |
|------|------|---------|
| **main_advance** | 主升浪进行中 | 当前上升浪评分 ≥ 历史最强主升浪×80% |
| **strong_advance** | 强势推进 | 当前上升浪振幅≥8 且 斜率≥1.5/期 |
| **pre_main** | 主升前布局 | 当前下跌中，前升浪≥5，RS未跌破前低-2 |
| **terminal** | 末浪衰竭 | 主升已过，当前上升浪力度<主升×60% |
| **recovery** | 弱势反弹 | RS<0，当前上升浪振幅<8 |
| **post_main** | 主升已过 | 默认状态 |

## 四、关键结论

1. **算法能识别主升浪** — 主升/强势信号后50日内最大涨幅均值+5.4，后浪均值17-22 RS点
2. **最有操作价值的是 pre_main** — 92%概率后续出>8RS大浪，且信号发出时处于调整/消化阶段，适合布局
3. **main_advance 是持有信号** — 发出时涨幅已大，后续虽有大浪但50日末均值为负（涨后回落）
4. **terminal 不一定是末日** — 末浪信号后仍有nan均值后浪，说明旧主升结束后新周期可能开始
5. **算法本质是波浪位置分类器**，按"你现在在哪段"指导操作，非短期择时
