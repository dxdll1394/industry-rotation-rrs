## 一、算法概要

| 步骤 | 方法 | 参数 |
|------|------|------|
| 趋势分段 | 5日采样轨迹 → 连续同向运动识别 | 最小2期(10天) |
| 连段合并 | 同向连续段自动合并为一个趋势 | — |
| 小波合并 | 逆势波振幅 < 周围主波×33% 则合并 | 33%阈值 |
| 主浪评分 | 振幅 × √交易天数 | ≥5振幅可参评 |
| 回撤过滤 | pre_main需满足30%~70%回撤率 | 30-70%区间 |

### 位置分类规则

| 阶段 | 条件 | 含义 |
|------|------|------|
| **main_advance** | 当前上升浪评分 ≥ 主浪×80% | 主升浪进行中 |
| **strong_advance** | 振幅≥8 且 斜率≥1.5/期 | 强势推进 |
| **pre_main** | 下降中 + 前升浪≥5 + 回撤30-70% + RS未破支撑 | 主升前布局 |
| **terminal** | 非强势 + 主升已过2浪 + 当前升浪<主浪×60% | 末浪衰竭 |
| **recovery** | RS<0 + 上升浪<8 | 弱势反弹 |
| **post_main** | 默认 | 等待下一轮 |

## 五、v2 vs v1 改进要点

| 改进 | 效果 |
|------|------|
| 连续同向段合并 | 消除分段碎片，减少误判 |
| pre_main 30-70%回撤过滤 | 排除极端回调（<30%太早，>70%太深） |
| terminal 不覆盖 strong_advance | 当前强势波不再被历史主浪压制 |
| start_val/end_val 传递 | 回撤率可精确计算 |
