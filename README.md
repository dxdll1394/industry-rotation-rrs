# 行业轮动 RRS 四象限图

基于相对强弱（Relative Strength）的 A 股行业轮动分析工具。

## 原理

**RS Ratio** — 行业/个股相对大盘的强弱，63 个交易日移动平均：
```
RS_Ratio = (价格 / 上证指数) 偏离其 63 日均线的百分比
```

**RS Momentum** — RS Ratio 的动量，21 日变化率：
```
RS_Momentum = RS_Ratio 的 21 日变化率
```

**四象限含义：**

| 象限 | RS Ratio | RS Momentum | 含义 |
|------|----------|-------------|------|
| L Leading | > 0 | > 0 | 强势领导 |
| I Improving | < 0 | > 0 | 改善转强（关注） |
| W Weakening | > 0 | < 0 | 开始走弱（警惕） |
| G Lagging | < 0 | < 0 | 持续低迷 |

### RS/MO 趋势组合判断

RRS 的核心是结合 RS 方向（连升/连降）和 MO 方向（正/负）判断趋势阶段：

| RS方向 | MO方向 | 含义 | 操作参考 |
|--------|--------|------|----------|
| ↑ | ↑（正） | 走强加速，最强势 | 持有/加仓 |
| ↑ | ↓（负） | 走强减速，优势收窄 | 警惕拐点 |
| ↓ | ↑（正） | 走弱逆转，趋势可能见底 | 观察/试仓 |
| ↓ | ↓（负） | 双降恶化 | 回避/减仓 |

**↑N/↓N** 标记表示 RS 连续上升/下降的期数。结合象限使用：
- **↑ + L/I** = 趋势加速信号
- **↓ + L/W** = 趋势减速预警

> 参考依据：该涨不涨为弱，该跌不跌为强。RS 方向变化比单期数值更重要——L 象限但 RS 连续下降比 I 象限但 RS 连续上升更危险。

### RS连续趋势统计（参考用）

基于 2023-06-06 至 2026-05-15 共 75 期（约3年）数据的统计规律，供预测时参考：

**行业层面（32个行业）：**
- 平均连升 1.5 期，连降 2.0 期
- 多数在 ↑1~↑2 之间往复锯齿
- ↑3+ 属罕见强势
- 下降分布比上升更分散，弱势行业更容易持续走弱

**逆转规律（610次行业级逆转）：**

| 连降 | 次数 | 占比 | 平均反弹 | 真逆转率 | 解读 |
|------|------|------|---------|---------|------|
| ↓1 | 315 | 51.6% | 3.5 | 15% | 单期噪声，近一半会弹回 |
| ↓2 | 144 | 23.6% | 3.4 | 18% | 正常波动，真逆转率最高 |
| ↓3 | **78** | **12.8%** | **4.0** | **10%** | **最佳逆转窗口** |
| ↓4 | 40 | 6.6% | 4.9 | 8% | 反弹尚可，但真逆转率骤降 |
| ↓5 | 18 | 3.0% | 5.9 | 11% | 样本少，偶有强反弹 |
| ↓6 | 10 | 1.6% | **7.9** | **0%** | 大反弹但必是死猫，无真逆转 |
| ↓7+ | 5 | 0.8% | 5.7 | **0%** | 极端情况，均无真逆转 |

**最大连降记录: ↓9期**

**核心规律：**
1. **↓1~↓3 占 88%** — 绝大多数连降在3期内解决
2. **↓2 是入场胜率最高的窗口**（真逆转率 18%），因为只是正常回调
3. **↓3 是平衡点**：反弹力度最大（均值4.0），仍有10%真逆转率，↓4后反弹质量断崖下降
4. **↓6+ 是陷阱**：虽然必然反弹且幅度大（均值7.9），但**100%是死猫反弹** — 没有一次能持续上涨。RS跌了6期以上的行业，基本面已经彻底恶化，反弹就是逃命机会
5. **↓4+ 整体真逆转率仅 6%**（5/73），说明连降超过3期后，即使反弹也不该相信是趋势反转

**应用原则：**
- 想抄底：等连降 **↓2~↓3** 时观察，配合 MO 转正信号
- 连降 **↓4+** 不要抄底，反弹即使大也是逃命波
- 持仓中连降 **↓3** 是最后的止损观察窗口，↓4+ 考虑止损

## 用法

```bash
python3 行业轮动RRS.py              # 只读缓存
python3 行业轮动RRS.py --force       # 强制刷新全部数据
python3 行业轮动RRS.py --html        # 同时生成交互式 HTML
python3 行业轮动RRS.py --cron        # 定时模式（仅周六更新）
```

输出：
- `数据/行业轮动RRS轨迹图.png` — matplotlib 静态图
- `数据/行业轮动RRS轨迹图.html` — ECharts 交互式图表

## HTML 交互功能

### 四象限图
- **快照回看** — 时间滑块选择历史日期，图表冻结到该时刻；勾选"对比当前"叠加当前位置
- **悬停高亮** — 跨系列联动（轨迹线 + 最新点 + 历史点）
- **单/多行业模式** — 单击图例切换，点击节点自动隔离
- **缩放** — 滚轮缩放、框选放大（右上角工具箱）
- **轨迹方向** — 箭头指示节点间运动方向（单行业时显示）
- **图例按象限排序** — 颜色 + 前缀标识（[L]/[I]/[W]/[G]），快照时按历史象限重新着色

### 趋势变化表
- **多维度排序** — 支持按名称/最新RS/最新MO/RS趋势/象限/股票数量排序
- **点击日期列** — 按该日期 RS 值排序
- **点击行业名** — 跳转个股趋势表，自动筛选该行业
- **展开成分股** — 查看个股 RS/MO 数据、点击跳转东方财富
- **↑N/↓N 标记** — RS 连续上升/下降期数，悬停显示详情

### 个股趋势表
- 381 只个股 RS/MO 轨迹一览
- **搜索+筛选** — 按名称/代码搜索，按行业/象限过滤
- **排序** — 名称/代码/行业/最新RS/最新MO/RS趋势/象限
- **点击跳转** — 股票代码可点击打开东方财富个股页
- **行背景色** — 按最新象限着色（L蓝/I绿/W橙/G灰）

## 学习方法（流星交易体系相关知识点）

本工具的 RRS 轮动分析对应以下交易方法论要点：

### 强弱判断
- 该涨不涨为弱（W/G 象限但应转强不转）
- 该跌不跌为强（I 象限但该弱不弱）

### 分歧与一致
- 一致阶段：RRS 趋势加速（↑+L/I），适用于持有/加仓
- 分歧阶段：RRS 趋势减速（↓+L/W），警惕拐点
- 分歧天数 ≈ 后面一致行情天数

### 风格轮动规律
```
科技 → 金融 → 资源/涨价 → 红利
```
用 RRS 可跟踪各风格所处的象限和趋势阶段，在风格启动前识别 I→L 的切换信号。

### 主线确认方法论
- 上涨时间窗口才有主线（L 象限行业集群出现时）
- 通过 5-7 天走势验证展望 → 确认上涨趋势 → 锁死主线
- 结合 RRS 趋势方向 + 行业属性综合判断

### 仓位参考
- 大盘不好（多数行业在 G 象限）：仓位 ≤ 30%
- 大盘一般（L/I 象限行业增多）：仓位 65-70%
- 大盘看好（主线明确，L 象限扩散）：仓位 90%
- 单行业仓位不超过 30%

## 依赖

- Python 3.9+
- pandas, numpy, matplotlib
- akshare（数据源）

## 数据

- 来源：东方财富（akshare `stock_value_em` 接口）
- 基准：上证指数（本地 SQLite 缓存）
- 缓存：个股收盘价 Parquet 文件（7 天增量更新）
- 自选股池：`数据/综合股池.json`（手动编辑）
