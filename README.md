# 行业轮动 RRS 四象限图

基于相对强弱（Relative Strength）的 A 股行业轮动分析工具。

## 原理

**RS Ratio** — 行业相对大盘的强弱，63 个交易日移动平均：
```
RS_Ratio = (行业等权平均价 / 上证指数) 偏离其 63 日均线的百分比
```

**RS Momentum** — RS Ratio 的动量，21 日变化率：
```
RS_Momentum = RS_Ratio × 21 日变化率
```

**四象限含义：**

| 象限 | RS Ratio | RS Momentum | 含义 |
|------|----------|-------------|------|
| L Leading | > 0 | > 0 | 强势领导 |
| I Improving | < 0 | > 0 | 改善转强 |
| W Weakening | > 0 | < 0 | 开始走弱 |
| G Lagging | < 0 | < 0 | 持续低迷 |

## 覆盖行业

30 个申万一级行业，158 只成分股，个股等权聚合。

石油石化、煤炭、有色金属、钢铁、化工、建材、工程机械、航运港口、金融（银行+证券+保险）、农牧、食品饮料、医药生物、电子、计算机、通信、电力设备、光伏、汽车、汽车零部件、家电、房地产、建筑、电力、交通运输、社服消费、军工、传媒、轻工制造、纺织服装、环保

## 用法

尽量少用强制刷新
```bash
python3 行业轮动RRS.py              # 只读缓存
python3 行业轮动RRS.py --force       # 强制刷新数据,慎用
python3 行业轮动RRS.py --html        # 同时生成交互式 HTML
python3 行业轮动RRS.py --cron        # 定时模式（仅周六更新）
```

输出：
- `数据/行业轮动RRS轨迹图.png` — matplotlib 静态图
- `数据/行业轮动RRS轨迹图.html` — ECharts 交互式图表

## HTML 交互功能

- **悬停高亮** — 跨系列联动（轨迹线 + 最新点 + 历史点）
- **单/多行业模式** — 单击图例切换，节点点击自动隔离
- **缩放** — 滚轮缩放、框选放大（右上角工具箱）、放大/缩小按钮
- **轨迹方向** — 箭头指示节点间运动方向（仅单行业时显示）
- **图例按象限分组** — 颜色 + 前缀标识（[L]/[I]/[W]/[G]）

## 依赖

- Python 3.10+
- pandas, numpy, matplotlib
- akshare（数据源）
- ECharts 5（HTML 渲染，CDN 加载）

## 数据

- 来源：东方财富（akshare `stock_value_em` 接口）
- 基准：上证指数（本地 SQLite 缓存）
- 缓存：个股收盘价 Parquet 文件
