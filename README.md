# 行业轮动 RRS 四象限图

基于相对强弱（Relative Strength）的 A 股行业轮动分析工具。

## 原理

**RS Ratio** — 行业/个股相对大盘的强弱，63 个交易日移动平均：
```
RS_Ratio = (价格 / 上证指数) 偏离其 63 日均线的百分比
```

**RS Momentum** — RS Ratio 的 21 日动量（差值）：
```
RS_Momentum = RS_Ratio[t] − RS_Ratio[t−21]
```

**四象限含义：**

| 象限 | RS Ratio | RS Momentum | 含义 |
|------|----------|-------------|------|
| L Leading | > 0 | > 0 | 强势领导 |
| I Improving | < 0 | > 0 | 改善转强（关注） |
| W Weakening | > 0 | < 0 | 开始走弱（警惕） |
| G Lagging | < 0 | < 0 | 持续低迷 |

### RS/MO 趋势组合判断

RRS 的核心是结合 RS 方向（连升/连降）和 MO 方向（正/负）判断趋势阶段：

| RS方向 | MO方向 | 含义 | 操作参考 |
|--------|--------|------|----------|
| ↑ | ↑（正） | 走强加速，最强势 | 持有/加仓 |
| ↑ | ↓（负） | 走强减速，优势收窄 | 警惕拐点 |
| ↓ | ↑（正） | 走弱逆转，趋势可能见底 | 观察/试仓 |
| ↓ | ↓（负） | 双降恶化 | 回避/减仓 |

**↑N/↓N** 标记表示 RS 连续上升/下降的期数。结合象限使用：
- **↑ + L/I** = 趋势加速信号
- **↓ + L/W** = 趋势减速预警

> 参考依据：该涨不涨为弱，该跌不跌为强。RS 方向变化比单期数值更重要——L 象限但 RS 连续下降比 I 象限但 RS 连续上升更危险。

### RS连续趋势统计（参考用）

基于 2023-05-30 至 2026-05-15 共 147 期（约3年，每5个交易日采样）数据的统计规律。

**核心：按实际交易天数分析（消除采样间隔影响）**

| 下降天数 | 逆转率 | 均反弹 | 解读 |
|---------|--------|-------|------|
| 0~9日 | 9.7% | 2.3 | 噪声期，不值得操作 |
| **15~24日** | **16.3%** | **3.4** | **★ 最佳逆转窗口（3~5周）** |
| 25~34日 | 12.3% | 3.7 | 仍可关注 |
| 35~44日 | 9.5% | 4.8 | 逆转率下降 |
| 45~54日 | 10.1% | 5.7 | 死猫反弹高发区 |
| 75日+ | **0%** | 6.2 | 零逆转，放弃 |

**间隔5日下的期数换算：**
- ↓3 = **15个交易日**（最佳窗口，16%逆转率）
- ↓5 = 25个交易日
- ↓8+ = 40日+，逆转率0%

**应用规律：**
1. **15~24个交易日（↓3~↓5）** 是逆转率最高的窗口，配合MO转正可入场
2. **10日以内** 是噪声，单期变化不必过度解读
3. **35日以上** 逆转率持续下降，超过**75日**零逆转
4. **45日以上** 反弹即使出现也是逃命波

详见 `docs/RRS趋势逆转分析报告.html`

## 主升浪分析算法

### 原理

基于 RRS 轨迹的波浪分解，识别行业所处的波浪阶段，寻找主升浪线索。

### 算法三步

**1. 趋势分段**
从 RRS 轨迹（5日采样）识别连续同向运动段。每段至少持续 2 期（10 个交易日），过滤单期噪声。

**2. 层级合并**
小幅逆势波（振幅 < 周围主波振幅 × 33%）被合并到周围的大趋势中。例如：大上升浪中的小幅回调不计入独立下跌段，合并为上升浪的一部分。这消除了日常波动噪声，保留了主要趋势结构。

**3. 主浪评分**
上升浪按 `振幅 × √交易天数` 评分。该公式奖励既大幅又持久的上升浪，避免将短期脉冲式反弹误判为主升浪。每个行业的历史最高分即为该行业的最强主升浪。

### 波浪位置分类

| 阶段 | 含义 | 判断条件 |
|------|------|---------|
| **main_advance** | 主升浪进行中 | 当前上升浪评分 ≥ 历史最强主升浪 × 80% |
| **strong_advance** | 强势推进 | 当前上升浪振幅 ≥ 8 且斜率 ≥ 1.5/期 |
| **pre_main** | 主升前布局 | 当前下跌中，前升浪 ≥ 5，RS未跌破前低 −2 |
| **terminal** | 末浪衰竭 | 主升已过，当前上升浪力度 < 主升 × 60% |
| **recovery** | 弱势反弹 | RS < 0，当前上升浪振幅 < 8 |
| **post_main** | 主升已过 | 默认状态，等待下一轮波浪结构 |

### 回测结果

基于 32 个行业、672 次滚动信号：

| 信号类型 | 后浪振幅均值 | 后浪>8胜率 | 操作含义 |
|---------|-----------|----------|---------|
| main_advance | 21.7 | 100% | **持有信号**，已在大浪中 |
| strong_advance | 17.7 | 97% | 持有，距主升还差空间 |
| **pre_main** | 17.8 | **92%** | **最佳布局信号** |
| terminal | 16.9 | — | 旧主升结束，等新周期 |
| post_main | 16.5 | — | 同上 |

核心结论：
- **pre_main 最有操作价值** — 92% 概率后续出现 > 8 RS 点的大浪，且信号发出时处于调整/消化阶段
- **main_advance 是持有/确认信号** — 发出时涨幅已大，不宜追高
- 算法本质是**波浪位置分类器**，按"你现在在哪个浪段"指导操作，不是短期择时工具

详见 `docs/主升浪算法回测报告.md`

### 方法论对应（流星交易体系）

| 算法概念 | 流星法则 |
|---------|---------|
| 趋势分段（同向 ≥ 2期=10天） | 法则13：日线3-5天/周线3-5周一致 |
| 层级合并（<33%振幅合并） | 法则22：三浪合一，小浪合并为大浪 |
| pre_main（下跌中前升浪≥5） | 法则6：abc行为后大概率加速 |
| main_advance（最强评分80%） | 法则19：第三笔高点高于第二笔 |
| terminal（主升后力度减半） | 法则8：急跌后震荡修复必破前高点 |
| 评分 = 振幅 × √天数 | 法则17：上涨目标为突破前杀多点 |

## 学习方法（流星交易体系相关知识点）

本工具的 RRS 轮动分析对应以下交易方法论要点：

### 强弱判断
- 该涨不涨为弱（W/G 象限但应转强不转）
- 该跌不跌为强（I 象限但该弱不弱）

### 分歧与一致
- 一致阶段：RRS 趋势加速（↑+L/I），适用于持有/加仓
- 分歧阶段：RRS 趋势减速（↓+L/W），警惕拐点
- 分歧天数 ≈ 后面一致行情天数

### 风格轮动规律
```
科技 → 金融 → 资源/涨价 → 红利
```
用 RRS 可跟踪各风格所处的象限和趋势阶段，在风格启动前识别 I→L 的切换信号。

### 主线确认方法论
- 上涨时间窗口才有主线（L 象限行业集群出现时）
- 通过 5-7 天走势验证展望 → 确认上涨趋势 → 锁死主线
- 结合 RRS 趋势方向 + 行业属性综合判断

### 仓位参考
- 大盘不好（多数行业在 G 象限）：仓位 ≤ 30%
- 大盘一般（L/I 象限行业增多）：仓位 65-70%
- 大盘看好（主线明确，L 象限扩散）：仓位 90%
- 单行业仓位不超过 30%
